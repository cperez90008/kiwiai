# SKILL: Calendar & Scheduling

## Without calendar access
- Help user think through scheduling conflicts
- Draft meeting invite text
- Suggest optimal meeting times based on described schedule
- Create time-blocking plans

## With Google Calendar MCP
- Check free/busy time
- Create events with details
- Find next available slot
- List upcoming events for a date range

## Best practices to suggest
- 15-minute buffer between meetings
- Deep work blocks in the morning
- Admin tasks in afternoon
- Protect Fridays for focused work
