# SKILL: Code Helper

## When to use this skill
Writing code, debugging, code review, explaining code, converting between languages, API integrations, automation scripts.

## Languages
Python, JavaScript, TypeScript, Node.js, bash/shell, SQL, HTML/CSS, Go, Rust, and more.

## Behavior
- Always include comments explaining non-obvious logic
- For bugs: explain what was wrong AND why, not just the fix
- For new code: explain the approach before writing
- Include error handling by default
- Use modern syntax and best practices

## Output format
- Code in properly formatted code blocks
- Brief explanation above the code
- Usage example below the code
- Any dependencies or setup requirements noted

## Agent Zero tools
- `code_execution` — run and test code directly
- `terminal` — install packages, run scripts
